[1.] PULSE DETECTOR

A pulse detector detects the pulse from your wrist displays the pulse rate and plots the graph of your pulse.
It uses the following components:
1.esp32: The microcontroller used for this project is esp32 because of its powerful features and compact size which makes it easy to use.
2.sensors: 3 infrared sensors are used for sensing the 3 different pulses from your wrist: Vata, Pitta, and Kapha.
3.Processing(Software): this software was used for making and integrating GUI with the hardware.
4. Arduino IDE: this was used for programming the esp32 board.

[3.] WEIN BRIDGE OSCILLATOR WITH PEAK-TO-PEAK DETECTOR

The components used are:
1.Wein bridge oscillator circuit: The Wien bridge oscillator is an audio frequency sine wave oscillator of high stability and simplicity.
The feedback signal in this circuit is connected to the non-inverting input terminal so that the op-amp is working as a non-inverting amplifier.
Therefore, the feedback network need not provide any phase shift.
2.Peak-To-Peak Detector Circuit: Peak detector circuits are used to determine the peak (maximum) value of an input signal.
It stores the peak value of input voltages for an infinite time duration until it comes to a reset condition.
The peak detector circuit utilizes its property of following the highest value of an input signal and storing it.
